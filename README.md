## What is MMSx-SCAN™?

MMSx-SCAN™ (Systematic Clinical Assessment Network) is a validated
multidimensional movement intelligence assessment system developed by
**Dr. Neeraj Mehta, PhD** at the MMSx Authority Institute for Movement
Mechanics & Biomechanics Research, Columbus, Ohio, USA.

It produces a composite **Movement Intelligence Index (MII, 0–100)**
across seven biomechanical domains plus a biomechanical flaw penalty
system (BFIS™).

**Published inter-rater reliability: ICC(2,1) = 0.955**
exceeding published FMS values (0.61–0.88).

## The 7 Domains

| Domain | Abbreviation | Max Points |
|--------|-------------|------------|
| I — Structural Integrity | SAS | 60 |
| II — Neuromechanical Intelligence | NIQ | 50 |
| III — Breath & Energetic Regulation | BES | 50 |
| IV — Performance Function | PRI | 50 |
| V — Load Intelligence | LIS | 50 |
| VI — Rehabilitation Velocity | RVS | 50 |
| VII — Blueprint Precision | EIS | 50 |

**MII Formula:**
MII = [(SAS+NIQ+BES+PRI+LIS+RVS+EIS) / 360] x 100 - BFIS Penalty
